# Gunzing_Opd.js

# Baseline Features: Evidence Identification System with Image Processing
ระบุวัตถุพยานด้วยเทคโนโลยีการประมวลผลภาพ


1. Model GUN
  - ตรวจจับอาวุธได้ 4 class: {glock19, cz75, sig365, colt1911}
  - หากจับภาพได้จะเข้าหน้าแสดงผล
  - แสดงผล: ชื่อ, โมเดล, ซีรี่ย์ของปืน, ขนาดกระสุนที่ใช้, ภาพปืนตัวอย่าง

2. Model Drug
  - ตรวจจับยาเสพติด 2 ประเภท: {WY หางสั้น, WY หางยาว}
  - แสดงผล: ชื่อยาเสพติด, ประเภทหลัก, ประเภทย่อย, พิกัด, รูปภาพ

4. Interface
  Login กรอกชื่อไอดี และ กรอกรหัสผ่าน เพื่อเข้าสู่ระบบการใช้งาน จากนั้นจะเข้าหน้า Home
  Home หน้าหลักของเว็บไซต์ มี sidebar และปุ่มถ่ายหรืออัปโหลดรูปภาพ
  Upload photo / camera ทำการถ่ายภาพหรืออัปโหลดภาพจากอุปกรณ์
  History ประวัติการประมวลผล แสดงเป็นรายการ
  Summary case  สรุปข้อมูล
  Profile / setting หน้าโปรไฟล์ของผู้ใช้ และหน้าตั้งค่า
  Sidebar เมนูนำทางไปยัง หน้าหลัก, ถ่ายรูป, กลุ่มงานอาวุธปืน, กลุ่มงานยาเสพติด

# User Flow

เริ่มต้นการใช้โดยการเปิดแอปพลิเคชั่น: ผ่านทางกรอกรหัสเข้าสู่ระบบ
พอเข้าหน้า Home จะสามาถ Upload รูปภาพ ที่ต้องการตรวจสอบ โดยจะสามารถใช้ได้ทั้ง Upload photo หรือ Camara
จากนั้นระบบจะแสดงผลข้อมูลภาพที่ตรวจจับได้
หากต้องการดูข้อมูลที่ถ่ายหรือตรวจสอบทั้งหมดแล้วสามารถเข้าไปดู Summary caseได้
